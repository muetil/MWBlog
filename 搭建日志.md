## 2021.11.07

### 引入标准化CSS

* 组件引入外部CSS

**全局,main.js内**

```js
import '@/assets/css/normalize/normalize.css'
```

**局部,组件内**

```html
<style scoped>
    @import "src/assets/css/normalize/myStandrad.css";
</style>
```

### 个人用标准化样式

```css
body,html{
    margin: 0;
    padding: 0;
}

a,a:hover,a:link,a:visited,a:active,a:focus{
    text-decoration: none;
    color: black;
}
```
